
# Week 1 - Day 3

## 📝 Overview

Day 3 focused on practicing intermediate SQL concepts using a custom Employee database.  
The session mainly covered filtering techniques, grouping operations, logical operators, comparison operators, pattern matching, and advanced query conditions.

The objective of this practice was to improve SQL problem-solving skills and strengthen understanding of data retrieval techniques using different SQL clauses and operators.

---

# 🎯 Objectives

- Practice intermediate SQL queries
- Work with filtering and grouping operations
- Understand logical and comparison operators
- Learn DISTINCT and TOP queries
- Practice pattern matching using LIKE operator
- Improve SQL query writing confidence
